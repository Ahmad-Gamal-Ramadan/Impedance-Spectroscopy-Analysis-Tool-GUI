## 🔬 Impedance Spectroscopy Analysis Tool

### 📌 Overview
A Python-based graphical user interface (GUI) application for analyzing electrochemical impedance spectroscopy (EIS) data. The tool enables users to load experimental data, visualize impedance spectra, and export high-quality plots for research purposes.

---

### ⚙️ Features
- Load impedance data files (frequency, Z', Z'')  
- Nyquist plot visualization  
- Bode magnitude plot  
- Bode phase plot  
- Display all plots simultaneously  
- Export plots as PNG, PDF, or SVG  
- Simple and user-friendly GUI  

---

### 🧪 Applications
- Electrochemical characterization of nanomaterials  
- Analysis of impedance response  
- Battery, supercapacitor, and sensor research  
- Academic and research use in materials science  

---

### 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- Matplotlib  
- NumPy
